# BlobAnywhere   

A browser extension that brings playful physics simulations to any webpage.

## Features

### 🫧 Blob Menu
Spawn customizable soft-body blobs directly onto any page. Watch them jiggle, bounce, and interact with physics in real-time. Tweak their properties and create your own squishy ecosystem.

### 🐛 Worm Lab
Release AI-controlled worms that squiggle and wiggle across your screen. These procedurally-animated creatures navigate and explore any webpage with simple, satisfying physics-based movement.

## How It Works

BlobAnywhere adds two interactive GUIs to any webpage:
- **Blob Menu** — Spawn and customize soft-body physics objects
- **Worm Lab** — Generate and observe AI-driven worms with emergent behavior

Just install the extension and start playing with physics on the internet.

## Installation

1. Clone or download this repository
2. Go to `chrome://extensions/` (or your browser's extensions page)
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder
5. Start spawning blobs and worms!

## Files

- `manifest.json` — Extension configuration
- `index.html` — UI interface
- `physics.js` — GUI and Physics
